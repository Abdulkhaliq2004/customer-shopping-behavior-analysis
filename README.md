# Customer Shopping Behavior Analysis

A simple end-to-end data analytics project to understand how customers shop across **demographics, categories, and seasons** using **Python, SQL, and Power BI**.

## Project Highlights
- Explored customer shopping data in **Python (Pandas)** and answered key business questions using **SQL**
- Built an **interactive Power BI dashboard** to track revenue trends, customer segments, and category performance
- Summarized findings in a short report and presentation

## Tech Stack
- **Python** (Pandas)
- **SQL** (queries + database-ready workflow)
- **Power BI** (dashboard)
- **Dataset**: CSV

## Folder Structure
- `notebooks/` → Python analysis (EDA)
- `sql/` → SQL queries
- `dashboard/` → Power BI `.pbix`
- `reports/` → PDF report + PPT
- `data/` → dataset (CSV)

## How to Run (Quick)
1. Open the notebook in `notebooks/`
2. Run Python EDA steps
3. Use `sql/` queries for analysis
4. Open Power BI file from `dashboard/`
